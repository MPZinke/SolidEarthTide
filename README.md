## Use

### Compiling

```bash
make
```

### Running

```bash
./SolidEarthFlexing
```
Or with input file
```bash
./SolidEarthFlexing < input.txt	
```

### Testing
```bash
git checkout Testing
make test
./Testing/SolidEarthFlexing < ./Testing/TestInput.txt
```
